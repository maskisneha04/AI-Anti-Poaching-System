AI-Based Anti-Poaching Wildlife Monitoring System

Smart surveillance to protect wildlife using Artificial Intelligence

Project Overview

This project uses AI and computer vision to monitor forest areas, detect poachers and wildlife activity, and alert forest officials in real time.

AI Detection

Detects humans, animals, and suspicious activity using AI models.

Live Monitoring

Real-time surveillance through cameras and uploaded videos.

Instant Alerts

Sends alerts to officials when poaching activity is detected.
